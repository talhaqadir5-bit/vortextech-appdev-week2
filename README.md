# To-Do List App

A simple and responsive To-Do List application built with **Flutter** as part of the **Vortex Tech App Development Internship – Week 2** assignment.

## 📱 Project Overview

This application demonstrates the basics of Flutter state management using `setState()`. Users can add new tasks, mark tasks as completed, and delete tasks through a clean and user-friendly interface.

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Strikethrough effect for completed tasks
- Delete tasks
- Responsive and clean UI
- Local state management using `setState()`

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Android Emulator or Physical Android Device

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/vortextech-appdev-week2.git
```

2. Navigate to the project directory

```bash
cd vortextech-appdev-week2
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
